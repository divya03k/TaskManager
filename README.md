# To-Do List App

A simple and interactive To-Do List application built with HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed or pending, and delete them with a smooth fade-out animation.

## Features

* Add new tasks
* Mark tasks as **Completed** or **Pending**
* Delete tasks with animation
* Clean and user-friendly interface

## Installation

1. **Download or clone** the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Navigate into the project folder:

   ```bash
   cd your-repo-name
   ```
3. **Open `index.html`** in any web browser.

No additional setup or installations are required.

## Usage

1. Type a task in the input field.
2. Click **Add Task** or press **Enter**.
3. Use the task buttons (**Pending**, **Completed**, **Delete**) to manage your list.

## Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)
